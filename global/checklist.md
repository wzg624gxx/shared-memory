## 常见问题和解决方案

### 编译问题
- [ ] 头文件找不到 → 检查 TC_ROOT/include 路径
- [ ] 库文件找不到 → 检查 TC_ROOT/lib 路径
- [ ] 链接错误 → 检查库文件顺序和依赖

### 运行时问题
- [ ] 登录失败 → 检查用户名/密码/组/角色
- [ ] 权限不足 → 检查用户权限和访问控制
- [ ] 对象不存在 → 检查 Tag 是否有效
- [ ] 内存泄漏 → 检查 MEM_free 调用

### 性能问题
- [ ] 批量操作使用数组 API
- [ ] 避免频繁的对象刷新
- [ ] 使用缓存减少重复查询
- [ ] 优化 BOM 遍历算法
