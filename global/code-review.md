# Teamcenter 13 代码审查

审查 Teamcenter 13 二次开发代码，检查以下方面：

## 检查清单

### ITK 代码
- [ ] 内存管理：所有 ITK 返回的 char* 是否使用 MEM_free 释放
- [ ] 错误处理：每个 ITK 调用后是否检查 status
- [ ] 事务管理：是否正确使用 ITK_exit_module
- [ ] API 版本：是否使用 TC13 推荐 API（而非已弃用 API）
- [ ] 资源释放：BOM 窗口、查询结果等是否及时释放

### SOA 代码
- [ ] 异常处理：是否捕获 ServiceException
- [ ] 连接管理：Connection 是否正确管理
- [ ] 批量操作：是否使用数组 API 而非循环
- [ ] 属性加载：是否先 getProperties 再读取

### BMIDE 配置
- [ ] 命名规范：是否使用公司前缀避免冲突
- [ ] 属性类型：字符串长度是否合理
- [ ] 规则逻辑：条件是否过于复杂

### Handler 代码
- [ ] 返回值：是否正确返回 EPM_go/EPM_nogo
- [ ] 附件处理：是否正确获取目标附件
- [ ] 错误消息：是否使用 EMH_store_error_s1 提供清晰错误
